# Implementation Plan: Special Foods & Event Card Manager

**Branch**: `002-special-foods-event-manager` | **Date**: 2026-04-11 | **Spec**: [spec.md](spec.md)
**Input**: Feature specification from `specs/002-special-foods-event-manager/spec.md`

## Summary

Extend the existing Snake game (Phaser 4.0.0, vanilla JS, single `game.js`) with 6 special food types and an event-card-based food spawning system. The current single `state.food` is replaced by `state.foods[]` (array of FoodItem). A new `EventCardManager` class orchestrates which food combinations appear, maintains a 10-card queue, tracks per-card telemetry, and delegates card generation to a standalone `fetchNextCard(telemetry)` function ready for future server integration. FOOD-BOMB introduces permanent obstacle cells (`state.obstacles[]`) that trigger game-over on head collision. A 32 px card-strip panel is appended below the grid (CANVAS_H: 600 → 632) to display the current card's remaining foods.

## `fetchNextCard(telemetry)` Adapter

Single standalone function at the top of `game.js`, before any class. Swapping this function is the only change needed for server integration.

```text
fetchNextCard(telemetry):
  1. Pick card size via CARD_SIZE_WEIGHTS cumulative wheel (1–4)
  2. For each slot: pick type via RARITY_WEIGHTS cumulative wheel
  3. Enforce: max 1 PENTA per card; max 1 BOMB per card (reroll if violated)
  4. Return { slots: string[] }
```

## Visual Food Rendering

All drawing done inside `drawFoodShape(gfx, food, cx, cy)` — called from `redraw()`:

| Type | Color | Phaser 4 draw call |
|------|-------|---------------------|
| STANDARD | `0xffeb3b` | `fillRoundedRect(cx-11,cy-11,22,22,5)` |
| DOUBLE | `0x00bcd4` | `fillPoints([N,E,S,W], true)` — 4-pt diamond |
| PENTA | `0xffc400` | `fillPoints(star10pts, true)` — 5-point star (10 verts, alt r=12/6) |
| TRIM | `0xff6d00` | `fillRoundedRect(cx-12,cy-5,24,10,5)` — flat wide pill |
| RUSH | `0xaa00ff` | `fillTriangle(cx,cy-12, cx+10,cy+8, cx-10,cy+8)` — upward triangle |
| STAR | `0xffffff` | `fillPoints(star8pts, true)` — 4-point star + yoyo alpha tween |
| BOMB | `0xb71c1c` | `fillCircle(cx,cy,10)` + two `strokeLineShape` X lines |
| OBSTACLE | `0x455a64` | `fillRect` + two `strokeLineShape` X lines |

`cx = food.x*CELL + CELL/2`, `cy = HUD_H + food.y*CELL + CELL/2` (pixel center of cell)

## Game-Loop Tick Changes

```text
Updated tick() pseudocode:

1. Apply nextDir → dir
2. Compute newHead
3. Wall collision?   → gameOver()
4. Obstacle collision? state.obstacles.some(o => match newHead) → gameOver()
5. Self collision?   → gameOver()
6. Food collision? Loop state.foods for match:
   - call applyFoodEffect(food)
   - remove food from state.foods
   - cancel food.starTimer if exists
   - call cardManager.onFoodConsumed(food)
7. if growthRemaining > 0: growthRemaining-- (skip pop)
   else: snake.pop()
8. snake.unshift(newHead)
9. Win check
10. redraw()
```

## FOOD-RUSH Speed Boost

```text
activateRush():
  rushActive = true
  boosted = max(TICK_MIN, floor(baseDelay / RUSH_SPEED_FACTOR))
  restartTick(boosted)                  ← tickGen++
  if rushTimerRef: rushTimerRef.remove(true)
  rushTimerRef = delayedCall(5000, rushExpired)

rushExpired():
  rushActive = false
  rushTimerRef = null
  restartTick(baseDelay)                ← uses score-updated baseDelay

updateSpeed() (score-based, unchanged trigger logic):
  newBase = max(TICK_MIN, baseDelay - TICK_STEP)
  baseDelay = newBase
  if NOT rushActive: restartTick(newBase)
  // if rushActive: baseDelay updated; running tick stays boosted; rushExpired() will use new baseDelay
  // Recreate FOOD-STAR timer(s) if any exist (baseDelay changed)
```

## FOOD-STAR Autonomous Movement

```text
createStarTimer(food):
  food.gen = state.tickGen
  food.starDir = random DIRS value
  food.starTimer = time.addEvent({
    delay: baseDelay * STAR_SPEED_FACTOR,
    callback: () => moveStar(food, food.gen),
    loop: true
  })

moveStar(food, capturedGen):
  if capturedGen !== state.tickGen: return  (stale, ignore)
  try directions in order [food.starDir, ...3 others shuffled]:
    if newPos is not wall/snake/obstacle/other-food:
      food.starDir = dir; food.x = newPos.x; food.y = newPos.y
      redraw(); return
  // all blocked → teleport
  food.x = randomFreeCell.x; food.y = randomFreeCell.y
  redraw()
```

## EventCardManager

```text
init():
  fill queue with 10 cards via fetchNextCard(telemetry)
  activateCard()

activateCard():
  cardDef = queue.shift()
  freeCells = getFreeCells()
  place foods: for each slot in cardDef.slots:
    if freeCells.length > 0: create FoodItem at random free cell; push to state.foods
    if STAR: createStarTimer(food)
  activeCard = { slots, remaining: placed count, drawnAt: Date.now() }
  appendNewCard()      ← keeps queue at 10
  updateCardStrip()
  redraw()

onFoodConsumed(food):
  activeCard.remaining--
  updateCardStrip()
  if remaining === 0:
    recordTelemetry()
    activateCard()

recordTelemetry():
  push { cardSlots, timeToConsume, snakeLength, score }
  if telemetry.length > 10: telemetry.shift()
```

## Card Strip HUD

Rendered at `y = HUD_H + ROWS*CELL` to `y = CANVAS_H` (y=600–632).

- Background: `fillStyle(C_HUD_BG)` + `fillRect(0, 600, CANVAS_W, CARD_STRIP_H)`
- For each remaining food type in activeCard: draw a mini food icon (12×12 px) and a count badge
- Uses a `this.cardStripGfx` (separate Graphics layer, depth > game gfx) + `this.cardStripTexts[]` (Phaser Text objects)
- Updated by `updateCardStrip()` — clears and redraws on each food consumption and card activation

## Implementation Phases

### Phase 1 — Constants & Drawing System (blocks all other phases)

1. Add `FOOD_TYPES`, `FOOD_COLORS`, `C_OBSTACLE`, `RARITY_WEIGHTS`, `CARD_SIZE_WEIGHTS`, `RUSH_BOOST_DURATION`, `RUSH_SPEED_FACTOR`, `STAR_SPEED_FACTOR`, `CARD_STRIP_H` constants; update `CANVAS_H` to `HUD_H + ROWS*CELL + CARD_STRIP_H` (632)
2. Write `function drawFoodShape(gfx, food, cx, cy)` — switch on food.type, draw correct shape for all 7 types using Phaser 4 Graphics API
3. Write `function drawObstacle(gfx, obs)` — dark slate filled square with two stroke X lines

### Phase 2 — Card Generation

4. Write `function fetchNextCard(telemetry)` — weighted random card using RARITY_WEIGHTS + CARD_SIZE_WEIGHTS; enforce max-1-PENTA and max-1-BOMB constraint; return `{slots}`
5. Write `class EventCardManager` with all methods: `constructor`, `init`, `activateCard`, `onFoodConsumed`, `recordTelemetry`, `appendNewCard`, `getFreeCells`, `destroy`

### Phase 3 — Food Effects & Snake Mutations

6. Write `GameScene.applyFoodEffect(food)` — dispatch effect by food.type
7. Write `GameScene.growSnake(n)` — `state.growthRemaining += n`
8. Write `GameScene.shrinkSnake(n)` — remove `min(n, snake.length-1)` tail segments
9. Write `GameScene.activateRush()` — boost tick, start 5s countdown; handle re-eat refresh
10. Write `GameScene.rushExpired()` — restore `state.baseDelay` tick; update STAR timer(s) if present
11. Write `GameScene.bombEffect()` — extract last segments, push to `state.obstacles`

### Phase 4 — FOOD-STAR Movement

12. Write `GameScene.createStarTimer(food)` — Phaser timer at `baseDelay × STAR_SPEED_FACTOR`
13. Write `GameScene.moveStar(food, capturedGen)` — directional movement with fallback teleport; gen-guard
14. Write `GameScene.cancelAllStarTimers()` — iterate `state.foods`, remove all starTimers; called on gameOver/gameWon

### Phase 5 — GameScene Refactor

15. Update `create(data)`: replace `food` → `foods:[]`; add `obstacles:[]`, `baseDelay`, `growthRemaining`, `rushActive`, `rushTimerRef`, `tickGen`; remove `spawnFood()` call; create `this.cardManager = new EventCardManager(this)`; call `cardManager.init()`; create `this.cardStripGfx` and `this.cardStripTexts`
16. Update `tick()`: replace single food check with `state.foods` loop; add obstacle collision; replace `growing`/`state.growing` with `growthRemaining` counter; call `cardManager.onFoodConsumed(food)` on hit
17. Update `updateSpeed()`: update `state.baseDelay`; only call `restartTick(newBase)` when not rush-active; recreate STAR timers if any STAR foods active
18. Write helper `GameScene.restartTick(delay)`: increment `state.tickGen`, remove old `tickRef`, create new `tickRef` with new delay

### Phase 6 — Card Strip Rendering

19. Write `GameScene.updateCardStrip()`: clear `cardStripGfx`; draw strip background; for each food type remaining in activeCard draw a 12 px mini-icon + count text
20. Call `updateCardStrip()` from: `cardManager.activateCard()`, `cardManager.onFoodConsumed()`, and `create()` (initial state)

### Phase 7 — Rendering & Lifecycle

21. Update `redraw()`: extend grid/food drawing to call `drawFoodShape` per food item in `state.foods`; add `drawObstacle` loop over `state.obstacles`; Phaser game config height already uses `CANVAS_H` so auto-resizes
22. Update `gameOver()` and `gameWon()`: call `cancelAllStarTimers()`; call `cardManager.destroy()`; set `this.cardManager = null`; clear rush timer if active
23. Re-validate Phaser.Game config: `height: CANVAS_H` — already references the constant, no inline change needed
